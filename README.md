# Description
  Gift Bliss takes the guess work out of buying and receiving great gifts

# UI
- User Profile Form
- Special Occassion Listing
- User See Individual Profiles of their friends

# Feature
- Users can create a profile
- Users can add friends to their profiles
- Users can see a list of upcoming special occasions for their friends
- Users can create a wish list or  a link to their amazon wishlist
- User can create a Google Calendar reminder for a special occasion
